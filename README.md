# Проект: Обработка HTTP запросов

##  Как запустить / открыть

Для запуска необходимо:

1. Склонировать репозиторий.
2. Выполнить в терминале: docker compose up 
3. После запуска открыть в браузере http://localhost:8000/

## Страницы
URL-адреса:

Главная страница (список новых вопросов) – /
Список лучших вопросов – /hot/
Список вопросов по тегу – /tag/{tag_name}/
Страница одного вопроса с ответами – /question/{question_number}/
Форма создания нового вопроса – /ask/
Форма входа – /login/
Форма регистрации – /signup/
Страница редактирования профиля – /profile/

Для просмотра неавторизованных страниц необходимо нажать (log out) рядом
с иконкой пользователя на верхней панели.

## Структура проекта:

- EDU-Web
    - application/ – настройки Django (settings.py, urls.py)
    - questions/ – приложение с функционалом вопросов и ответов
        - templates/questions/ - шаблоны для приложения questions
        - static/questions/ - статика для приложения questions
    - manage.py – скрипт управления Django
    - static/ – собранные статические файлы (CSS, JS, картинки)
    - Dockerfile – инструкция для сборки образа
    - docker-compose.yml – для запуска через Docker
    - requirements.txt – зависимости проекта
- .gitignore – исключения для Git

## Примечание

В рамках ДЗ созданы все view с данными-заглушками («рыба»), настроена именованная маршрутизация URL, шаблоны реализованы через наследование base.html и include, реализована функция пагинации с обработкой PageNotAnInteger и EmptyPage, подключён шаблон пагинатора, а также настроено рабочее окружение: requirements.txt, .gitignore для Django/Python/IDE, Dockerfile и docker-compose.yml.
