# Проект: вёрстка веб-приложения (VK Education)

## Как запустить / открыть вёрстку

Для просмотра необходимо:

1. Склонировать репозиторий.
2. Открыть HTML-файлы в `public/`.

## Страницы 

HTML-файлы из `public/`:

- **Главная страница (список вопросов)** – `public/index.html`
- **Страница одного вопроса с ответами** – `public/question.html`
- **Форма создания нового вопроса** – `public/ask.html`
- **Форма входа** – `public/login.html`
- **Форма регистрации** – `public/signup.html`
- **Страница редактирования профиля** – `public/profile.html`

## Структура проекта

- `public/` – все HTML-страницы
- `public/static/css/` – стили (Bootstrap + свои)
- `public/static/js/` 
- `public/static/img/` – изображения, аватарки, иконки

## Примечание

В текущей версии реализована только статическая вёрстка без бэкенда. Переключение состояний авторизированного/неавторизированного пользователя показано закомментированными блоками.
Глянуть пример неавторизированного пользователя можно на странице sign up.
